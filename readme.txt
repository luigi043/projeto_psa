Tecnologias/Frameworks Utilizados
Decisões de Design

Arquitetura MVC: Optei por seguir o padrão arquitetural Model-View-Controller (MVC) para garantir uma separação clara de responsabilidades no código.
Eloquent ORM: Utilizei o Eloquent ORM, fornecido pelo Laravel, para interagir com o banco de dados de maneira simplificada e eficiente.
Blade Templating Engine: Adotei o Blade como mecanismo de templates para estruturar as visualizações do lado do servidor.
Desafios Encontrados e Soluções
D: Integração com Banco de Dados

S: O Eloquent ORM facilitou a interação com o banco de dados, e segui as melhores práticas do Laravel para garantir a segurança e eficiência das consultas.
D: Roteamento e URLs Amigáveis

S: Utilizei o sistema de roteamento do Laravel para criar URLs amigáveis, garantindo uma navegação intuitiva e fácil compreensão.
D: Layout e Estilização

S: Implementei um layout simples e responsivo utilizando CSS, Bootstrap, ou qualquer outra ferramenta de estilização adequada para garantir uma experiência consistente ao usuário.
